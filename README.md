# Starter

`starter` branch

I have two apps as starter

- express app (server)

- vite react app (client)

# tRPC installs

server:

```
npm i @trpc/server zod --workspace="my_server"
```

client:

```
npm i @trpc/client @trpc/server @trpc/react-query @tanstack/react-query --workspace="my_client"
```
